# ArcEngine
C++ Cross Platform Engine to make personal games

## Supported Platforms
- Current Supported
    - Windows (OpenGL)
    - MacOSX (Metal)
- Future Support
    - Linux (Vulkan)

## Current Features:
- Windows
    - Window Creation Using OpenGL
- MacOSX
    - Window Creation Using Cocoa
    - Rendering Basic Triangle Using Metal Graphics API
## Future Features
- Platform Agnostic
    - Renderer (2D to start)
    - Input System
